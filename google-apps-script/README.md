# 記名多選投票：Google Sheet + Apps Script

這個版本使用固定名單的記名多選投票。每次送出都會記錄到 `NamedVotes` 分頁；同一名稱最後一次送出的選項才會計入總票數。

## 一次性設定

1. 開啟目標 Google Sheet，從網址 `/d/` 與下一個 `/` 之間複製試算表 ID。
2. 在該試算表選擇「擴充功能 → Apps Script」。
3. 用本資料夾的 `Code.gs` **完整覆蓋**原本程式碼，並把第一行 `SPREADSHEET_ID` 改成你的試算表 ID。
4. 儲存後選擇「部署 → 管理部署作業 → 編輯（鉛筆）→ 新版本 → 部署」。首次或權限變更時，依畫面授權試算表存取。
5. 網頁中的 `VOTE_API_URL` 應保留這個部署的 `/exec` 網址。若重用了既有部署，網址不會改變。

## 查看結果

- 公開網頁會顯示各民宿的票數、已投票名單與尚未投票名單。
- Google Sheet 會自動建立 `NamedVotes` 分頁，內有每次送出的原始紀錄：時間、姓名與勾選的民宿 ID。
- 原本的 `Votes` 分頁不會被修改，舊的測試紀錄會保留。

## 規則與限制

- 可投票名單固定為：哥布林、小埋、傑哥、企鵝、白白、CH、黑熊、叫我、雪糕、yuuu、久保。
- 一人可勾選一間以上民宿；重新送出會完整覆蓋該名字的舊選擇。
- 這不是登入系統：它能防止同名重複計票，但無法阻止知道別人名稱的人冒名送出或覆蓋投票。

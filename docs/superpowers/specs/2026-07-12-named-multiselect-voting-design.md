# 記名多選投票設計

## 目標

將包棟選擇頁改成 11 位固定成員的記名多選投票，顯示每間住宿被多少人選擇，以及誰已投、誰尚未投。

## 成員名單

哥布林、小埋、傑哥、企鵝、白白、CH、黑熊、叫我、雪糕、yuuu、久保。

## 使用流程

1. 投票者由下拉選單選擇自己的名字。
2. 投票者以核取方塊選擇一間以上的住宿。
3. 送出後，以名字為唯一身分；同名再次送出時，最後一次選擇覆蓋前一次整組選擇。
4. 頁面顯示每間住宿的不同投票人數、已投名單與未投名單。

## 資料與後端

Google Sheet 的 `Votes` 工作表每列保存時間、名字與多選住宿 ID 清單。Apps Script 只接受固定名單的名字、至少一個有效住宿 ID；讀取結果時依名字只採用最後一列，計算各住宿票數與投票狀態。

## 限制

不使用 Google 登入或個人密碼。任何持有連結的人仍可能冒用他人名字，但系統會阻止同名被重複計票。每次修改 Apps Script 後需更新既有網頁應用程式部署。

## 驗證

靜態檢查確認頁面包含名單、多選欄位與名稱提交。端到端測試以一位成員送出兩個住宿、確認兩邊各加一票；再以相同名字覆蓋成另一組選擇，確認舊選擇不再計票且已投名單只出現一次。
